# Installation

## Requirements

Using Composer:

    $ composer require visto9259/laminas-webpack

Or, alternatively, manually add the following to your `composer.json`, in the `require` section:
````json
{
  "require": {
    "visto9259/laminas-webpack" : "^1.0"
  }
}
````
